In order to run the project.
There are some important steps you should need to follow.

1.Install the node in your system and set a env for node.js
2.Go to frontend folder and run command npm i(this command will get all the packages from packages.json and install all the depenedncies in the node modules folder)
3.Run the command expo start.(if above command npm i not working than run yarn install).
4.After node_modules installation hit the command expo start.
5.In order to run the app in your phone locally you have to install the App
in your device called Expo Go.
6.signup in Expo Go in your device and login with the same credentials in command shell terminal.
7.Then you have to hit the command  Expo start than you will able to run the project.
